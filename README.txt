# Twitter Backend Clone 🐦

A RESTful backend for a Twitter-like platform built using Node.js and PostgreSQL.

## 🚀 Live API
https://twitter-backend.onrender.com

## 🧪 Sample Routes
- `POST /api/register`
- `POST /api/login`
- `POST /api/tweet`
- `GET /api/feed`

## 🛠️ Tech Stack
- Node.js
- Express.js
- PostgreSQL
- JWT Auth
- Render for deployment

## 📦 How to Run Locally
```bash
git clone https://github.com/yourname/twitter-backend
cd twitter-backend
npm install
cp .env.example .env
npm start
